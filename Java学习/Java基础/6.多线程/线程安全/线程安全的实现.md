#### Lock接口：非块结构同步

Lock提供了更广泛的锁定操作，ReentrantLock为其实现类

Lock接口中的方法：

- `lock()`：获取锁

- `unlock()`：释放锁

**步骤**：

- 在类中创建ReentrantLock对象
- 在可能出现线程安全问题的代码前调用lock()，获取锁
- 在代码结束位置调用unLock()，释放锁（最好放在**finall语句块**中，无论是否有异常都会释放锁）
