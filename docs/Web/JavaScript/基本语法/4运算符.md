### 1.一元运算符

#### ++：自增

#### --：自减

#### +(正号)、-(负号)

若运算数不是运算符所要求的类型，那引擎会自动进行类型转换，转换为number类型

- string 转number：按字面值转换，若字面值不是数字，则转换为NaN
- boolean转number：true转为1，false转为2

### 3. 逻辑运算符

#### &&

若左边为false：则不再运算

#### ||

短路效果：若左边为true，则不再运算

#### ！

其他类型转为boolean：

- number转boolean：0为false，其他为true
- string转boolean：""为flase，其他为true
- null或undefined都为false
- 对象都为true
