#### 1、显示当前文件状态

`git status`

- 查看工作区文件的状态：是否修改

#### 2、版本比较

`git diff`

- 显示工作区内容和暂存区内容的区别

`git diff --cached `

- 查看暂存区和仓库的区别

`git diff HEAD -- filename`

- 查看工作区和仓库最新版本的区别

#### 3、查看提交历史

`git log`

- 查看当前分支版本历史

`git log`

- 查看当前分支版本历史

`git log --oneline`

将提版本历史按行输出

`git log -<number>`

- 显示指定数量的版本历史

`git log --graph`

以图形界面显示版本历史
