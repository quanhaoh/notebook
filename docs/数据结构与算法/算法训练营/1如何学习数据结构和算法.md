#### 五遍刷题法

1. 第一遍
   - 5分钟看题——思考
   - 5分钟想不出马上看解法，背诵+默写

2. 第二遍
   - 不看答案自己写一遍

3. 第三遍
   - 24小时后重新练习该题
4. 第四遍
   - 1周后进行专题练习
