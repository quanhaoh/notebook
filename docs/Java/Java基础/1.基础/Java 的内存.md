#### 1、栈（Stack）
存放方法的局部变量，且方法的运行一定是在栈中运行

#### 2、堆（Heap）
凡是 new 出来的东西，一定在堆中

不同new 出来的东西在堆中的位置不同，但可以被栈中的同一个引用变量先后引用

堆内存中的数据都有默认值，根据数据类型不同，默认是也不同

#### 3、方法区（Method Area）
存储 class 相关信息，包含方法的信息
存储类的静态变量/方法

#### 4、本地方法栈（Native Method Stack）
与操作系统有关

#### 5、寄存器（Register）
与CPU有关
